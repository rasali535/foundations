# Foundations Counselling Academy

A web application for the Foundations Counselling Academy, a Pameltex Group company.

## Overview

"Workplaces where people thrive — measurably."

This repository contains the frontend React application for Foundations Counselling Academy. It provides information on their services such as Corporate Training, Psychosocial Risk Management, EAP & Counselling, and Organisational Development.

## Development Setup

This frontend is built with React and Tailwind CSS. 

### Prerequisites

- Node.js
- Yarn (Recommended) or npm

### Available Scripts

Navigate to the `frontend` directory:

```bash
cd frontend
```

#### `yarn start` or `npm start`

Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

#### `yarn build` or `npm run build`

Builds the app for production to the `build` folder.
It correctly bundles React in production mode and optimizes the build for the best performance.
